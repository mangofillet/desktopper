Lamp now: `SpotLight(0xffa95c, 4.2, 3.2, 0.62, 0.7, 1.4)`.

Gotchas learned this session:
- The classic_laptop glTF loads async (~2-4s); early screenshots show NO laptop — NOT a bug. probe.mjs now waits 5.2s; verify waits 5.2s before the hero shot. Laptop meshes + OS screen have frustumCulled=false (re-parented screen plane gave a bad bounding sphere).
- GitHub Pages `deploy-pages` step intermittently returns "Deployment failed, try again later" (error_count:10) — a GitHub-side outage, not our code. Just re-run `gh workflow run deploy.yml` until it passes.
